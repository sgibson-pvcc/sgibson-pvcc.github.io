TERM DEFINITION
LAN normally consists of a collection of computers in a single building or building complex
WAN links machines over a greater distance—perhaps in neighboring cities or on opposite sides of the world.
MAN normally consists of a collection of computers in a single building or building complex
Bus Network is a network that is not connected to a hub to transmit data
Star Network has a hub to transmit data, therefore the network takes a longer time to process because it has to go through the hub first.
CSMA/CD Ethernet a bus network based on the Ethernet standards, the right to transmit messages is controlled by the protocol
CSMA/CA Wifi wireless networks adopt the policy of trying to avoid collisions rather than trying to detect them
repeater a device that passes signals back and forth between the two original buses without considering the meaning of the signals
bridge looks at the destination address that accompanies each message and forwards a message across the connection only when that message is destined for a computer on the other side
switch a bridge with multiple connections
router(GUI) connection between networks to connect to the internet
gateway the point at which one network is linked to an internet
p2p: peer to peer client/server model involves one process (the server) providing a service to numerous others (clients), the peer-to-peer model involves processes that provide service to and receive service from each other
distributed systemsoftware units that execute as processes on different computers.
cloud computing virtual computing resources available on the network
tier 1 ISPs that consist of very high-speed, high-capacity, international WANs.
tier 2 ISPs that tend to be more regional in scope and less potent in their capabilities. (The distinction between the tier-1 and tier-2 ISPs is often a matter of opinion.) these networks tend to be operated by companies in the communications business.
tier 3 procedure that transfers the operating system from mass storage (where it is permanently stored) into main memory (which is essentially empty when the machine is first turned on)
ICANN a nonprofit corporation established to coordinate the Internet’s operation. ICANN
DNS Domain Name System are name servers that are used as an Internet-wide directory system
TLD: Toplevel domain Classification of domain names according to the suffix
Country code TDLs Classification of domain names according to the country location. World Standards
FTP File Transfer Protocol (FTP), or an application for accessing another computer from a great distance used the Telnet protocol, or later, the Secure Shell (SSH) protocol.
HTTP traditional network applications have come to be handled by webpages via the powerful Hyper Text Transfer Protocol (HTTP)
SMPT defines a way that two computers on the network may interact when transmitting an email message from one host to the other.
URL contains the information needed by a browser to contact the proper server and request the desired document
Client-side activities a webpage involving animation or one that allows a customer to fill out an order form and submit the order? These needs would require additional activity by either the browser or the webserver. These are called client-side activities if they are performed by a client (such as a browser)
Server-side activities activities that are performed by a server (such as a webserver).
Application layer consists of those software units such as clients and servers that use Internet communication to carry out their tasks
transport layer adds sequence numbers to the small segments it produces so that the segments can be reassembled at the message’s destination
network layer layer to decide in which direction a packet should be sent at each step along the packet’s path through the Interne
link layer has the responsibility of transferring the packet. The link layer must deal with the communication details particular to the individual network in which the computer resides.
Packet-switching network transmits data across digital networks by breaking it down into blocks or packets for more efficient transfer
Fault tolerant the ability of a system to continue operating without interruption when one or more of its components fail.
Redundant most or all of the network remains connected even when very important tier routers fail.
Malware software that can be transferred to, and executed on, the computer itself, or it might attack the computer from a distance.
virus software that infects a computer by inserting itself into programs that already reside in the machine. Then, when the “host” program is executed, the virus is also executed.
worm s an autonomous program that transfers itself through a network, taking up residence in computers and forwarding copies of itself to other computers. As in the case of a virus, a worm can be designed merely to replicate itself or to perform more extreme vandalism
Trojan horse a program that enters a computer system disguised as a desirable program, such as a game or useful utility package, that is willingly imported by the victim. Once in the computer, however, the Trojan horse performs additional activities that might have harmful effects. Sometimes these additional activities start immediately. In other instances, the Trojan horse might lie dormant until triggered by a specific event such as the occurrence of a preselected date. Trojan horses often arrive in the form of attachments to enticing email messages.
spyware software that collects information about activities at the computer on which it resides and reports that information back to the instigator of the attac
phishing a technique of obtaining information explicitly by simply asking for it. The term phishing is a play on the word fishing because the process involved is to cast numerous “lines” in hopes that someone will “take the bait.” Phishing is often carried out via email, and in this form,
DDos atack when the signal is given, all of these computers swamp the target with messages. When the work of generating the attacks is distributed over many hosts
firewall filters traffic passing through a point in the network
spam filter firewalls designed for specific purposes, for example, to block unwanted email
proxy server is a software unit that acts as an intermediary between a client and a server with the goal of shielding the client from adverse actions of the server.
antivirus software software used to detect and remove the presence of known viruses and other infections.
cryptography the study of sending and receiving messages securely in the presence of adversaries
public key encryption a technique by which the encryption and decryption algorithms are known, and one of the keys is known, but the encoded message remains secure.
